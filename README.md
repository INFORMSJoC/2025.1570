## Description

This repository contains the Python implementation of the robust data-driven quasiconcave optimization framework presented in the paper. The code includes:

1. **Quasiconcave Interpolation Methods**: 
   - Worst-case quasiconcave interpolation with Lipschitz constraints
   - Piecewise constant interpolation
   - Concave regression

2. **Robust Optimization Algorithms**:
   - Binary search algorithm for robust quasiconcave optimization
   - Level function method (MILP-based baseline)

3. **Applications**:
   - Cobb-Douglas utility optimization
   - Max-min fairness optimization for facility location

## Replicating Results

### Cobb-Douglas Utility Experiments (Section 6.1)

To run all experiments for the Cobb-Douglas utility function:

```bash
python scripts/run_experiments.py --experiment all
```

Individual experiments can be run separately:

```bash
# Visualization of interpolation methods
python scripts/run_experiments.py --experiment visualize

# L1 error comparison across sample sizes
python scripts/run_experiments.py --experiment l1_error

# Optimality gap comparison
python scripts/run_experiments.py --experiment optimality_gap --n-rep 20

# Scalability of interpolation methods
python scripts/run_experiments.py --experiment scalability_interp

# Scalability of robust optimization
python scripts/run_experiments.py --experiment scalability_ro
```

### Max-Min Fairness Experiments (Section 6.2)

To run all max-min fairness experiments:

```bash
python scripts/run_maxmin_experiments.py --experiment all
```

Individual experiments:

```bash
# Visualization of max-min utility problem
python scripts/run_maxmin_experiments.py --experiment visualize

# L1 error comparison
python scripts/run_maxmin_experiments.py --experiment l1_error --n-rep 20

# Fairness analysis
python scripts/run_maxmin_experiments.py --experiment fairness --n-rep 200

# Computational efficiency comparison
python scripts/run_maxmin_experiments.py --experiment computational --n-rep 50
```

### Command Line Options

Both scripts support the following options:

- `--experiment NAME`: Run specific experiment (default: `all`)
- `--no-plots`: Disable plot generation
- `--n-rep N`: Number of replications for statistical experiments (default: 20)

## Results

Running the experiments will generate:

1. **Figures** in the `results/` directory:
   - `true_func.png`, `true_contour.png`: True objective function visualizations
   - `qco_int.png`, `const_int.png`, `co_reg.png`: Interpolation method comparisons
   - `L1.png`: L1 error vs sample size
   - `maxmin_*.png`: Max-min fairness experiment results

2. **CSV files** in the working directory:
   - `df_optimality_gap_*.csv`: Optimality gap results
   - `df_scalability_*.csv`: Computational scalability results
   - `df_maxmin_*.csv`: Max-min experiment results
